Core-Features of the Js:

  • Dynamically typed language.
    • doesn't have to mention the types of datatype we give

    • js is partially OOP's based, it only follows encapsulation, inheritance, polymarphism.
  • funtional type programming.
    • js follows the functional way of programming which follows call backfunctions, higer order functions and closures.
  • First-class Functions
    • Functions in js are first-class why because, they can be passed as arguments and they can return another function and they can be store in data structures.
  • Prototype based inheritance
